# HTML:
<h1> para titulos <P> para a escrita no site <nav> para criar um menu e se mover pelas paginas,junto ao <li> para organizar os objetos,<ul> para organizar o menu tabem, <link rel> para liga o css e java aos html para personalizar as cores e fontes,<img> para colocar a imagens do jogo no site,<href> para localizar os arquivos html para o menu,<center> para centralizar os itens(texto,imgs,titulos) de forma pratica,<class> para colocar uma classe na <nav> para personaalizr o menu, <title> para o nome na aba de navegação

# CSS:
<background-color> define a cor de fundo, <color> define a cor do elemento, <border-radius> define o tamanho da borda, <border> define borda,<fontfamily> serve para personalizar a forte, <witht> define tamaho do elemento, <paddin> define grossura da linha da borda, <text decoration> retirar as decorações do item, <justify item> recofigurar a posição do item

# java
<const> usei para criar uma variavel constante para o formulario, <fuction> usei para criar as funções do formulario(enviar, e criar os espaços para as informações),<preventDefault> para previnir a funcionalidade padrão,<If>  <else> para decidir como agir em uma situação,<alert> pra enviar uma mesangem ao executar uma ação no site, <addEventListener> adiciona o evento ao site
